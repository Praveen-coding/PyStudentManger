# PyStudentManger
This is web application used Flask.

**To Use This Application You have to do these steps:-**<br/>
1 - Install Python2 or Python3<br/>
2 - Install git<br/>
3 - Install pip<br/>
```
apt-get install pip
```
4 - Install Flask
```
pip install flask
```
5 - Open `app.py`
```
python3 app.py
```
6 - Then you saw a server address.
```
http://127.0.0.1:5000/
``` 
Then paste this link into your browser.
